cymapgt\core\application\spreadsheet\SpreadsheetProcessor\SpreadsheetProcessor

PHPExcel_CachedObjectStorageFactory - This class encapsulates a number of key PHPExcel methods in order to make creation and manipulation of files less verbose - It also provides an organized structure of managing temporary files on disk.

- Extends PHPExcel capability by adding a SWF filetype to the stack of file formats
