cymapgt\core\application\spreadsheet\SpreadsheetProcessor\PHPExcelWrapper::autoConvert

Automatically convert a file type to another

Automatically convert a file type to another. (From CSV to XLS/XLSX and back) The purpose of this function is primarly to convert an XLS/XLSX file to a CSV file for ease of reading data (just open a file handle, read line by line and do an explode(',', $line)
