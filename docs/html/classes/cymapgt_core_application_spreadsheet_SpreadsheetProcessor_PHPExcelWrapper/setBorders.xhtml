Set the borders around cells in an Excel file

    Signature

    public function setBorders([integer $col = 0, [int $row = 0, [PHPExcelBorderType $borderSides = PHPExcelBorderType::All, [PHPExcelBorderStyle $borderType = PHPExcelBorderStyle::BORDER_THIN, [bool $isCoordinate = false]]]]] )

    Parameters

    $col — integer
    (OPTIONAL) The column to set the borders around.
    Default is 0. (0 says all columns)
    $row — integer
    (OPTIONAL) The row to set the borders around.
    Default is 0. (0 says all rows)
    $borderSides — object
    (OPTIONAL) The sides to set the border on.
    Default is PHPExcelBorderType::All.
    $borderType — object
    (OPTIONAL) The border style to set.
    Default is PHPExcelBorderStyle::BORDER_THIN.
    $isCoordinate — object
    (OPTIONAL) If this is false, the function expects
    $col to be a numeric value. If this value is true
    this funcition simply concatenates $col and $row
    Default is false.
