cymapgt\core\application\spreadsheet\SpreadsheetProcessor\PHPExcelWrapper::setHyperlink

Add a hyperlink to a cell

When adding a hyperlink to a cell it does not color and underline the cell as if you were in Excel, to emulate this, set $autoColor = true
