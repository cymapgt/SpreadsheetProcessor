cymapgt\core\application\spreadsheet\SpreadsheetProcessor\PHPExcelWrapper::setCellFont

Set the font name and size of a cell

If $col and $row are set to 0 (their default values), then the entire active sheet is set to the font family ans size

    Signature

    public function setCellFont(string $fontName, [int $fontSize = 0, [int $col = 0, [int $row = 0, [bool $isCoordinate = false]]]] )

    Parameters

    $fontName — string
    The font family name to set (e.g. 'Arial', 'Calibri', etc.)
    The font name must be a valid font name to set
    $fontSize — int
    The font size to set (e.g. 10, 12, etc.)
    $col — integer
    (OPTIONAL) The column to set the font on.
    Default is 0. (0 means all columns)
    $row — integer
    (OPTIONAL) The row to set the font on.
    Default is 0. (0 means all rows)
    $isCoordinate — object
    (OPTIONAL) If this is false, the function expects
    $col to be a numeric value. If this value is true
    this function simply concatenates $col and $row
    Default is false.
